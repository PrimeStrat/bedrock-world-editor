# World Editor

A Minecraft Bedrock tool that combines a bit of World Edit & Axiom. This tool is intended for fixing small issues in a build with similar tools without constantly switches versions.

## Getting started

1. Add both packs to your world and enable them.
2. Play in **Creative** as an **operator** (most commands are op-only).
3. Run `/we:book` for the in-game guide, or `/we:wand` to grab the selection
   wand.
